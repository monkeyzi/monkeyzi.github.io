java中的经典排序算法–选择排序

原理

选择排序：遍历元素找到一个最小（或最大）的元素，把它放在第一个位置，然后再在剩余元素中找到最小（或最大）的元素，把它放在第二个位置，依次下去，完成排序

时间复杂度

O(n^2)
第一次需要遍历n个元素，接着需要遍历此时分别为 n-1,n-2,n-3,…..2,1次
